C'est quoi un code glouton?

Un code glouton est un type d'algorithme qui permet l'optimisation de certains calculs.
Le principe d'un algorithme glouton est de faire les meilleurs choix possibles a chaque etape.
L'idee est qu'en faisant les meilleurs choix pour chaque etape on obtient le meilleur choix global.
Souvent un code glouton ne correspond pas a la meilleure strategie mais il peut produire des resultats locaux qui
peuvent ensuite etre utiliser pour obtenir un resultat global optimise.

Exemple

Pierre vend des journaux. Un jour il vend un journal pour 1,14 euros.
Le client paie avec une piece de 2 euros. Pierre veut minimiser le nombre de pieces redonnees en change.
Pour cela il va utiliser la methode gloutonne.
Le code pour faire cela est ci-dessous.

Code

Voici un code qui redonne la quantite de pieces qu'il faut donner pour obtenir une certaine valeur de centimes .
Supposant qu'un jour on veut acheter quelque chose, ce code va retourner les pieces necessaires pour payer les centimes qui restent.
Dans cette situation le code retourne les pieces qu'il faut avoir pour obtenir 86 centimes le plus effectivement.
